# 📓 Python Eğitim Notları (Interactive Notebooks)

Bu klasör, **BilgiSezer ile Sıfırdan Python Öğreniyorum** serisinin üniversite düzeyindeki tüm ders anlatımlarını ve uygulama dosyalarını içermektedir. 

### 🚀 Nasıl Kullanılır?
Bu dersler interaktif bir yapıda hazırlanmıştır. Öğrenme sürecinizi verimli kılmak için şu adımları izleyebilirsiniz:

1. **İncele:** İlgili konunun `.ipynb` dosyasına tıklayarak kodları ve açıklamaları GitHub üzerinden görüntüleyebilirsiniz.
2. **Uygula (Google Colab):** Her dosyanın en üstünde yer alan **"Open in Colab"** butonuna tıklayarak, bilgisayarınıza herhangi bir kurulum yapmadan kodları tarayıcı üzerinden çalıştırabilir ve üzerinde değişiklik yapabilirsiniz.
3. **Not Al:** Kendi kopyanızı oluşturarak ders notlarının üzerine kendi yorumlarınızı ekleyebilirsiniz.
