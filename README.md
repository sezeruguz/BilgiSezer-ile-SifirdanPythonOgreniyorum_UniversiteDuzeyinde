# 💻 BilgiSezer ile Sıfırdan Python Öğreniyorum (Üniversite Düzeyinde)

Bu repo, Python programlama dilini sıfırdan öğrenmek isteyenler için hazırlanmış bir eğitim serisini içerir.

Amaç; yalnızca bir programlama dili öğretmek değil, aynı zamanda algoritmik düşünme, problem çözme ve yazılım geliştirme becerileri kazandırmaktır.
